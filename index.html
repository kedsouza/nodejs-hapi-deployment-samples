Below are the api links methods

GET /api | (Lists all names)

POST /api | request body { name: "example_name" } | (Inserts name into db)

PATCH /api | request body { id : 1, name: "example name"} | (Edits name based on id)

PUT /api | request body { id : 1 (optional), name: "example name"} | (Overwrite id and name of entry, if entry does not exist creates new name)

DELETE /api | request body { id : 1} | (Deletes name based on ID)
