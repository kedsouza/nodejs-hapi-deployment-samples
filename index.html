Method Path Request Body Description
GET /api Lists all names
POST /api { name: "example_name" } Inserts name into db
PATCH /api { id : 1, name: "example_name"} Edits name based on id
PUT /api { id : 1 (optional), name: "example_name"} Overwrite id and name of entry, if entry does not exist creates new name
DELETE /api { id : 1} Deletes name based on id
